Synchronisez automatiquement la base des utilisateurs de votre site Wordpress avec une liste de destinataires sur la plateforme emailing MDWorks.

== Description ==

Ce plugin permet de synchroniser automatiquement la base des utilisateurs de son site sous Wordpress avec une liste de destinataires hébergée sur la plateforme emailing MDWorks.
Il est entièrement en français. Les fichiers de traduction sont disponibles.
Avec ce plugin, il est possible d'ajouter, de modifier ou de supprimer une entrée dans une liste de contacts hébergée sur la plateforme emailing MDWorks dès qu'une opération équivalente est effectuée dans la base des utilisateurs de Wordpress.
Cette extension est compatible avec les plugins de gestion des membres qui utilisent la base des utilisateurs Wordpress. Il est donc facile d'automatiser des envois d'email ou des scénarios d'envoi d'email lors de l'inscription d'un nouveau membre.


== Installation ==

1. Placez les fichiers dans le répertoire `/wp-content/plugins/mdworks-hosted-lists`, ou bien lancez l'installation à partir de la page des extensions Wordpress.
2. Activez le plugin à partir de la page des extensions Wordpress.
3. Rendez-vous sur la page Réglages->MDWorks pour configurer le plugin.

For english people :

1. Upload the plugin files to the `/wp-content/plugins/mdworks-hosted-lists` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->MDWorks screen to configure the plugin


== Frequently Asked Questions ==

= A qui s'adresse le plugin ? =

Cette extension est disponible gratuitement pour tous les utilisateurs de la plateforme email-marketing MDWorks. Il permet de synchroniser automatiquement la base utilisateurs Wordpress avec une liste sur MDworks. Il est compatible notamment avec les plugin de gestion de membres.
